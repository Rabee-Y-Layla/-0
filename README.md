# Interval Linear Regression for Tokamak Data

## Project Description
This project implements interval linear regression analysis for tokamak plasma inversion radius data. The program processes experimental data and builds interval regression models with uncertainty quantification.

## Files Structure

### Source Code
- `U-1.py` - Main Python script for interval regression analysis

### Input Data
- `normalised_export.csv` - Normalized tokamak experimental data
- `inversion radius .csv` - Inversion radius measurements

### Output Files
- `interval_regression.png` - Regression plot with intervals
- `regression_results.csv` - Complete analysis results

### Documentation
- `Отчет_Интервальная_регрессия_РАСШИРЕННЫЙ.docx` - Detailed report (in Russian)
- `README.md` - This file
